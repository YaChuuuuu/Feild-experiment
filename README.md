施用內生菌對草莓生長、抗逆境之影響 (三個處理組別、三個試驗區域)

2021 四月 至 2021 年九月，每個月收一次數據 (生長指標與病害指數)，一個處理組別 30 個植株 (3\*3\*30=270)
